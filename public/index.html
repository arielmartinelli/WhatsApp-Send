Recomendaciones de Seguridad

¡Muy Importante!

WhatsApp prohíbe el envío masivo de spam. Para cuidar tu cuenta:

  • Usa Delays Amplios Configura al menos 10-25 segundos entre mensajes en la pestaña de Consola.
  • Mensajes Personalizados Personaliza cada mensaje con el nombre del contacto. Envía textos diferentes usando variables.
  • Calienta el número No uses números nuevos de WhatsApp para enviar muchos mensajes. Deben tener conversaciones previas de ida y vuelta.
  • Prueba Primero Siempre haz una prueba enviando a tu propio número antes de iniciar una lista larga.
